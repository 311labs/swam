SearchDown Component

This component is used in replacement of a select/dropdown when trying to select an item from a large collection. It make it easy to search for remote data. You can either search locally or remotely. In your parent just implement on_action_searchdown to capture the change event!

Constructor Properties

  • all view properties - see view docs
  • collection - the SWAM.Collection instance to use
  • btn_classes - defines the look of the dropdown button
  • search_field - the name of the field to search on (default: search)
  • display_field - the name of the field to show in the list (default: name)
  • empty_label - what to show when nothing is selected
  • remote_search - enable remote queries for searching (default: false)
  • auth_fetch - auto fetch the collection on first click (default: false)
  • inline - an inline style for the dropdown
  • on_top - a popover style for the dropdown
  • List - optional List View Class
  • ItemView - optional Item View Class

Functions

  • setActive(model) - set the selected model
  • addChild(el_sel, view) - adds a child view

Events/Actions

This component will fire "searchdown" action. Thus it can be caught by implementing on_action_searchdown in any parent class.

Examples


        this.collection = new SWAM.Collection({
              debounce_time: 1000,
              url:"https://test.payauth.io/rpc/location/ips",
              size:20}),
        this.addChild("mysearchdown", new SWAM.Views.SearchDown({
              display_field: "ip",
              search_field: "search",
              remote_search: true,
              collection: this.collection, 
              title:"Select Item"}));
