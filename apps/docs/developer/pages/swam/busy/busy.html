Loading/Busy Animation

It is critical to inform the user when data is loading or the app is busy. This is made easy by the following helper methods.

Busy Overlay

Show busy overlay quickly and easily.

Code

SWAM.Dialog.showLoading();

Show inline busy over a component.

Code
SWAM.Dialog.showLoading({
    parent:this.$el.find("#test_busy")
});

Provide icons for more context.

{{#busy_examples}} {{{.|icon}}} {{/busy_examples}}
Code
app.showBusy({
  icon:kind,
  color:color,
  timeout:4000,
  no_timeout_alert:true
});
