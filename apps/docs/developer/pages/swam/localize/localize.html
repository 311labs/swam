SWAM.Localize Usage

Easily convert data to be more human readable.

{{=||| |||=}} 
// Localization is done normally in the mustache/html template.

<h1>{{model.size|bytes}}</h1>

// you can render right in JS as well
var txt = SWAM.renderString(
    "This was created: {{model.created|datetime}}", 
    {model:this.model});
    
// or localize it as we get it from the model
var when = model.get("created", null, "datetime");
