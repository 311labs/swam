Misc Examples

A collection of misc components.

Dropdown

Simple Dropdown via JS!


this.addChild("simple_dropdown", new SWAM.Views.Dropdown({
      menu_items: [
          "Blue", "Green", "Yellow",
          "Red", "Purple", "Black", 
          {id:"light", label:"White"}
      ]
  }));

this.children.simple_dropdown.on("change", this.on_dropdown_change, this);
Popover

Pure HTML popover component!

    
<button type="button" 
  class="btn btn-lg btn-danger" 
  data-bs-toggle="popover" 
  data-bs-trigger="focus" 
  title="Popover title" 
  data-bs-content="And here's some amazing content. It's very engaging. Right?">
Click to toggle popover
</button>
Accordian

This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.
